# Agent Skills

Portable skills and reusable context bundles for AI coding agents.

This repository is organized as one directory per skill:

```text
agent-skills/
  adonis-v6-backend/
    SKILL.md
    references/
  nuxt-3-frontend/
    SKILL.md
    references/
```

Each `SKILL.md` contains the core instructions that should load when the skill is used. Longer examples, conventions, workflows, and reference material should live in that skill's `references/` directory.
